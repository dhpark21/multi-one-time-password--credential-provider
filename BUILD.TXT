How to build:

- Open MultiOneTimePassword.sln in Visual Studio 2010

- Solution -> Rebuild all (it excludes the setup projects)


How to build Setup Project:

- Rebuild Solution in Release [Win32|{Eng, Ger}], [x64|{Eng, Ger}]

- Select Setup-x64 or Setup-x86

- The currently active configuration will be packaged into the installer, so

  : when running x64-installer-builds make sure a x64 platform is targeted and vice versa

  : the release language will be packaged

- Build the Setup Project
